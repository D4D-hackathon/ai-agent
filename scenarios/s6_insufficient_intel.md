# S6 — 첩보 불충분 (Insufficient Intel)

**작전목표:** 협로 확보를 위한 공격. 그러나 **적 정보가 불완전**하다.
**상황:** 아군은 협로 확보를 위해 공격을 준비하나, 적 전차 수·태세가 파악되지 않았다.
CLAUDE.md 원칙: 누락 필드는 **지어내지 말고 사용자에게 되묻는다**. 근거 불충분 시
"판단 근거 불충분"을 명시한다.

- 날씨: `fog`  · 지형: `ridge`
- 아군(Blue): 전차 10, 병력 400, 위치 (38.2450, 127.0800), 태세 `attack`
- 적군(Red):  전차 **미상**, 병력 **미상**, 위치 (38.2610, 127.0930), 태세 **미상**

## 교육 포인트
- `/scenario` 는 누락 필드(`적군.전차`, `적군.병력`, `적군.태세`)를 `422 + missing` 으로 되묻는다.
- 정보 획득(정찰) 전에는 확신에 찬 판정을 내리지 않는다.

## POST /scenario 페이로드 (의도적으로 불완전 → 422 되묻기 유도)
```json
{
  "objective": "안개 속 협로 확보 공격",
  "weather": "fog",
  "terrain": "ridge",
  "blue": {"tanks": 10, "troops": 400, "lat": 38.2450, "lon": 127.0800, "posture": "attack"},
  "red":  {"lat": 38.2610, "lon": 127.0930}
}
```
→ 응답: `422 {"missing": ["적군.병력","적군.전차","적군.태세"]}` (정찰로 채운 뒤 재요청)
